libpng integration
===============================================================================

Tilengine requires static libpng 1.6 to build. Put here the following files:

png.h
pngconf.h
pnglibconf.h
zconf.h
zlib.h
/linux/libpng16.a
/linux64/libpng16.a
/osx/libpng.a
/Win32/libpng.lib
/x64/libpng.lib
